Quantitative Finance ↗

Adaptive Factor Model with Regime Detection

Regime-Dependent Fama-French Factor Analysis · Python + R

  • Fits a Gaussian Hidden Markov Model on daily market returns to classify trading days into volatility regimes (low, medium, high), then estimates separate Fama-French + Momentum factor exposures per regime
  • Chow test confirms factor loadings are statistically different across regimes: momentum flips negative in high-vol states, market beta spikes to ~1.3, and value exposure doubles
  • Full residual diagnostics per regime using Newey-West HAC standard errors, Durbin-Watson, Ljung-Box, and Breusch-Pagan tests
  • Generates 5 publication-quality plots: regime-colored cumulative returns, factor loading comparisons with 95% CI, rolling Sharpe ratios, regime distribution, and annualized alpha by regime
  • Dual implementation in Python (hmmlearn, statsmodels, scipy) and R (depmixS4, lmtest, sandwich, ggplot2) with a built-in data simulator and 14-test pytest suite
Python R statsmodels hmmlearn scipy ggplot2 Newey-West HMM Fama-French pytest
